## Asian Recipe Classifier Case Study ##
A DS 4002 Case Study by Zoe Gates

### The Hook and Rubric ###
The hook document which introduces the case study is labeled "HOOK_Asian_Recipes_Classifier." It provides insight into the importance of this project and will outline the task and deliverables. The rubric titled "RUBRIC_Asian_Recipes_Classifier" delves deeper into the specific specifications categories and provides more details on the deliverables/formatting required.

### Data ### 
The data was scraped from https://tasteasianfood.com/. We have included a comprehensive CSV in the "Data" folder, so that you do not have to go through this. We used BeautifulSoup in order to scrape the data.
